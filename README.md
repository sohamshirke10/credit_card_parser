## Quick start
1. Backend:
   - cd backend
   - python3 -m venv venv
   - source venv/bin/activate
   - pip install -r requirements.txt
   - uvicorn app:app --host 0.0.0.0 --port 8000 --reload

2. Frontend:
   - cd frontend
   - npm install
   - npm run dev
